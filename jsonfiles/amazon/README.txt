The data in this directory came from:

http://jmcauley.ucsd.edu/data/amazon/

Per the maintainer's request, one of his publications should be cited, e.g.:

http://cseweb.ucsd.edu/~jmcauley/pdfs/kdd15.pdf

To avoid checking in a large amount of data to the repo, you must directly download the file linked as:

"Movies and TV	 5-core (1,697,533 reviews)"

Only the first 10,000 lines of this file are checked in to this repository.
